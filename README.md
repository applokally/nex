# ATL NEX Landing Page

Projeto estático responsivo para escolha de plano, chip físico/eSIM,
novo número/portabilidade e app de benefício, sem formulário.

## Rodar localmente

```powershell
python -m http.server 5500
```

Abra `http://localhost:5500`.

## Consultores

Edite `assets/js/consultores.js`.

O número de suporte/ativação dos materiais de referência não foi incluído.
